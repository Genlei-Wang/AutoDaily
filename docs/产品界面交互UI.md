
# AutoDaily 产品界面示意 (UI/UX Specification) - 优化版

## 1. 界面线框图 (Wireframe Description)

主窗口大小：**400px * 600px**（支持DPI自动缩放，适配150%、200%等系统缩放）。

```text
+-------------------------------------------------------------+
||  AutoDaily 日报助手                                    [ - ][ x ]|
+-------------------------------------------------------------+
||                                                             |
||   🟢 就绪  (状态指示灯，绿色/黄色圆形图标 + 文字)          |
||                                                             |
||   +---------------------------------------------------+     |
||   |  (A) 核心操作区 (Card) - 白色背景，圆角8px        |     |
||   |                                                   |     |
||   |   [ 🔴 录制 ]          [ ▶️ 运行 ]                |     |
||   |   (录制新动作)        (运行跑一遍)                |     |
||   |   红色边框按钮          蓝色填充按钮                |     |
||   |                                                   |     |
||   +---------------------------------------------------+     |
||                                                             |
||   +---------------------------------------------------+     |
||   |  (B) 定时运行 (Card) - 浅灰背景，圆角8px          |     |
||   |                                                   |     |
||   |   [ =======O ] 自动运行  (开关默认关闭)           |     |
||   |                                                   |     |
||   |   ────────────────────────────────────────       |     |
||   |   (以下内容仅在开关开启时显示)                      |     |
||   |                                                   |     |
||   |   每天 [ 09:00 ] 自动运行                         |     |
||   |   (时间选择器，仅显示时:分，不显示秒)               |     |
||   |                                                   |     |
||   |   *下次运行：12月05日 09:00                      |     |
||   |   (灰色小字提示)                                  |     |
||   +---------------------------------------------------+     |
||                                                             |
+-------------------------------------------------------------+
```

**布局说明：**
- **状态指示灯：** 左上角，20px 边距，12pt 粗体，绿色圆形图标 + "就绪"文字
- **核心操作区卡片：** 白色背景，360px 宽，120px 高，圆角 8px，20px 内边距
  - 录制按钮：160x60px，红色边框（2px），白色背景，红色图标和文字
  - 运行按钮：160x60px，蓝色填充（RGB: 0,122,204），白色文字
  - 提示文字：8pt，灰色（RGB: 150,150,150），位于按钮下方 5px
- **定时运行卡片：** 浅灰背景（RGB: 250,250,250），360px 宽
  - 关闭状态：50px 高，仅显示开关和"自动运行"标签
  - 开启状态：110px 高，显示时间选择器和下次运行时间
  - 开关：iOS 风格滑块，蓝色激活状态（RGB: 0,122,204）
  - 时间选择器：仅显示时:分格式（如 09:00），不显示秒，80x25px

### 悬浮录制条 (Overlay Bar) - 录制时出现，屏幕顶部居中

**设计原则：** 最小化遮挡，紧凑布局，深色半透明背景

```text
+---------------------------------------------------------+
|  🔴 录制中  00:12  |  [ ⏸ 暂停 ]  |  [ ⏹ 完成并保存 ]  |
+---------------------------------------------------------+
```

**尺寸：** 宽度 380px，高度 50px（支持DPI缩放，可优化至 350x45px 减少遮挡）
**位置：** 屏幕顶部居中，距离顶部 10px
**背景：** 深灰色半透明（RGB: 50,50,50，透明度85%），圆角 15px
**字体：** 白色，10pt 粗体
**按钮：** 扁平化设计，无边框，悬停高亮，70x30px
**优化建议：** 进一步缩小尺寸，减少对用户操作的遮挡
**优化建议：** 可进一步缩小至 350x45px，减少遮挡

## 3. 运行时的反馈设计 (The Feedback)

**屏幕提示:**
*   **小提示框 (HUD)**：
    ```text
    +---------------------------------------+
    |  [×]                                  |
    |  🤖 正在运行                          |
    |  ------------------------------------ |
    |  执行步骤 3/5: MouseClick             |
    |  [████████████░░░░░░░░] 60%          |
    |  ⚠️ 按 F10 停止                       |
    |  或点击右上角 × 关闭                  |
    +---------------------------------------+
    ```

**尺寸：** 宽度 280px，高度 120px（支持DPI缩放，已优化为更小尺寸）
**位置：** 屏幕右上角，距离右边缘 20px，距离顶部 20px
**背景：** 浅灰色（RGB: 240,240,240），白色内容面板，圆角 10px
**布局：** 
  - 顶部：关闭按钮（右上角 ×，30x30px）
  - 标题：🤖 正在运行（蓝色 RGB: 0,122,204，10pt 粗体）
  - 状态：执行步骤 X/Y: 动作类型（8pt，灰色 RGB: 60,60,60）
  - 进度条：绿色填充（RGB: 76,175,80），15px 高，显示百分比
  - 提示：按 F10 停止（橙色 RGB: 255,152,0，7pt）
  - 提示：或点击右上角 × 关闭（灰色 RGB: 150,150,150，6pt）
*   **目的**：明确告知用户"现在我是自动驾驶状态"，防止用户惊慌或误触，同时最小化遮挡。

## 2. 交互优化

### (A) 录制、运行操作区
*   **布局**：两个并排的大按钮，占据视觉中心。
*   **[🔴 录制]**：
    *   **设计**：红色边框（2px），红色图标，白色背景。
*   **[▶️ 运行]**：
    *   **设计**：蓝色填充（RGB: 0,122,204），白色文字。
    *   **防呆设计**：如果还没有录制过数据，这个按钮置灰不可点，提示文字变为红色"请先录制动作"。

### (B) 定时运行
*   **设计**：浅灰色背景的容器（RGB: 250,250,250），位于底部。
*   **交互**：
    *   **默认状态（关闭）：** 仅显示开关和"自动运行"标签，卡片高度 50px
    *   **开启状态：** 显示时间选择器（仅时:分格式）和下次运行时间，卡片高度 110px
    *   **开关**：iOS风格的滑块开关（Toggle Switch），蓝色激活状态
    *   **时间格式：** 仅显示时:分（如 09:00），不显示秒
