# AutoDaily 日报助手

一个轻量级、易用的桌面自动化工具，用于简化银行、政府等单位职员的日报流程。

## 功能特性

- 🎬 **动作录制**：录制鼠标点击和键盘操作
- ▶️ **自动执行**：回放录制的动作序列
- ⏰ **定时运行**：支持每天定时自动执行，支持"补卡"机制
- 🎨 **简洁界面**：参考"录音笔"设计理念，420x280精致窗口
- 🛡️ **安全可靠**：运行时遮罩层提示，F12紧急停止

## 技术规格

- **平台**: Windows 7/10/11 (x86/x64)
- **框架**: .NET Framework 4.7.2
- **UI**: WinForms
- **体积**: < 2MB (零第三方依赖)

## 项目结构

## 编译说明

1. 使用 Visual Studio 2017 或更高版本打开 `AutoDaily.sln`
2. 确保已安装 .NET Framework 4.7.2 开发工具
3. 编译项目（Release模式）
4. 输出文件位于 `AutoDaily/bin/Release/AutoDaily.exe`

## 使用说明

### 录制动作

1. 点击"🔴 录制"按钮
2. 开始操作目标应用程序（点击、输入等）
3. 操作完成后，点击悬浮条上的"⏹ 完成并保存"

### 运行任务

1. 点击"▶️ 运行"按钮
2. 软件会自动执行录制的动作序列
3. 如需停止，按 F12 键

### 定时运行

1. 设置定时时间（如 09:00）
2. 打开定时开关
3. 软件会在每天指定时间自动运行
4. 支持"补卡"机制：如果电脑锁屏，解锁后会自动补跑

## 数据存储

任务数据存储在：`%AppData%\AutoDaily\tasks.json`

## 注意事项

- 录制时请确保目标窗口处于活动状态
- 运行时会自动调整目标窗口大小以匹配录制时的尺寸
- 支持多显示器环境
- 建议在运行前关闭其他可能干扰的应用程序
