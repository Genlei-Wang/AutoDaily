# 贡献指南

感谢您对 AutoDaily 项目的关注！我们欢迎各种形式的贡献。

## 如何贡献

### 报告问题

如果您发现了 bug 或有功能建议，请通过 GitHub Issues 提交：

1. 检查是否已有类似的问题
2. 创建新 Issue，详细描述问题或建议
3. 如果是 bug，请提供：
   - 操作系统版本
   - 复现步骤
   - 预期行为 vs 实际行为
   - 相关日志（如有）

### 提交代码

1. Fork 本项目
2. 创建功能分支 (`git checkout -b feature/AmazingFeature`)
3. 提交更改 (`git commit -m 'Add some AmazingFeature'`)
4. 推送到分支 (`git push origin feature/AmazingFeature`)
5. 创建 Pull Request

### 代码规范

- 遵循 C# 编码规范
- 使用有意义的变量和函数名
- 添加必要的注释
- 确保代码通过编译
- 保持代码简洁和可读性

### 测试

在提交 PR 前，请确保：
- 代码可以正常编译
- 基本功能测试通过
- 没有引入新的 bug

## 开发环境

- Visual Studio 2017 或更高版本
- .NET Framework 4.7.2 SDK
- Windows 7/10/11

感谢您的贡献！
